# 🌱 GreenRecycle

## Project Description

GreenRecycle is a web-based recycling reward management system.

Customers can bring reusable or recyclable items to participating stores to earn reward points, which can later be redeemed for gifts.

The system helps stores manage recycling transactions efficiently while encouraging sustainable environmental practices.
